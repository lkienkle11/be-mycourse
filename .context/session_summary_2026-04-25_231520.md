## Key decisions
- Chốt kiến trúc media **stateless cloud gateway** thay vì local persistence — phù hợp yêu cầu nghiệp vụ mới.
- Setup SDK media ở startup (`main.go` -> `pkg/media.Setup`) để đồng nhất pattern với DB/config bootstrap.
- Tách trách nhiệm package:
  - `constants` cho enum/constants
  - `entities` cho shape dữ liệu
  - `dto` cho request/response contract
  - `pkg/media` cho SDK integration
  - `pkg/logic/helper` cho runtime guard + metadata/local token helpers
  - `services` chỉ orchestration business flow.
- Giữ error contract nhất quán theo `pkg/errcode`.
- Chấp nhận giữ historical context files để trace quá trình thay đổi, nhưng canonical docs phải khớp code hiện tại.
