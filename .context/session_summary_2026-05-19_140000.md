# Session: Logout API (BE)

**Date:** 2026-05-19

## Contract

- `POST /api/v1/auth/logout`
- Headers: `X-Refresh-Token`, `X-Session-Id` (same as refresh)
- Success: `200`, `message: logout_success`, `Set-Cookie` clears `access_token`, `refresh_token`, `session_id`
- Idempotent: missing session key → `200` + clear cookies
- `401` on UUID mismatch; cookies still cleared
